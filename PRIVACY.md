# Privacy Policy for Alignment Inspector

**Last updated: January 2026**

## Overview

Alignment Inspector is a browser extension that helps web developers and designers check element alignment on webpages. Your privacy is important to us.

## Data Collection

**Alignment Inspector does not collect, store, or transmit any user data.**

## How the Extension Works

- All functionality runs entirely within your browser
- No data is sent to any external servers
- No analytics or tracking of any kind
- No cookies are used

## Webpage Access

The extension only accesses webpage content locally to:
- Read element positions and dimensions
- Display visual alignment guides and measurements

This information is:
- Processed entirely in your browser
- Never stored persistently
- Never transmitted anywhere
- Cleared when you close the tab or disable the extension

## Permissions Explained

- **activeTab**: Allows the extension to access the current tab only when you click the extension icon
- **scripting**: Required to inject the alignment overlay into the webpage
- **host_permissions**: Allows the extension to work on any website you choose to inspect

## Changes to This Policy

If we make changes to this privacy policy, we will update the "Last updated" date above.
